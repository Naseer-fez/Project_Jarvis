## High-Level Identity

Jarvis is a modular local AI assistant. The project has a Python runtime, local/cloud LLM routing, memory, desktop automation, voice, dashboard UI, integrations, workflows, and safety controls.

## LLM Orientation Summary

When working on Jarvis, start at:

1. `main.py` and `main_connector.py` for the launcher surface.
2. `core/runtime/entrypoint.py` for startup, health checks, dashboard boot, and shutdown.
3. `core/controller_v2.py` for request routing and high-level orchestration.
4. `core/controller/services.py` for service construction and dependency wiring.
5. `core/agent/agent_loop.py` for plan, risk, execute, reflect flow.
6. `core/llm` for model routing and LLM calls.
7. `core/tools` and `integrations` for actions Jarvis can perform.
8. `tests` for expected behavior before changing code.
