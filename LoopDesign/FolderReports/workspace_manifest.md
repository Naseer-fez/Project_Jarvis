# Workspace Directory Manifest

## High-Level Purpose
The `workspace` directory functions as an active runtime and staging area for the Jarvis agent. It serves as a synchronization point and data exchange interface, utilizing a dropbox mechanism to ingest commands, perform Retrieval-Augmented Generation (RAG) processes, and store output artifacts such as UI screenshots and general test files.
