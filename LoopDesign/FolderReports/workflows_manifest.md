# Workflows Folder Manifest

## Overview
The `workflows` directory is designed to define, store, and manage structured sequences of tasks or automated pipelines within the Jarvis system.

## Contents

- **`templates/`** (Directory)
  - **Purpose**: Houses workflow templates which serve as blueprints for generating active workflow instances. Currently empty, but structurally set up for template storage.
  - **Required Specialists**: Configuration Analyst, Dependency Analyst, Prompt Recovery Specialist
