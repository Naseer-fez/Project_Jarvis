# File Report: config/settings.env.template

## Purpose
Provides a structural template for the `settings.env` file. Used by new users or CI/CD systems to understand which environment variables are required.

## Responsibilities
- Document expected secret keys and API configurations.
- Ensure developers know what fields are necessary to configure the system without exposing actual secrets.

## Architecture Role
Serves purely as documentation/template for environment configuration. Includes placeholders for Email, WhatsApp, Home Assistant, GitHub, Wake word, Cloud LLM Fallbacks, HuggingFace, and Web Search (Tavily).
